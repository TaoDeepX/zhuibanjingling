# 追板精灵（复刻版）

完整复刻追板精灵功能，实时监控A股异动并语音播报。

当前版本：**v1.7**

---

## 快速开始

双击运行 **`启动追板精灵.bat`**

---

## 功能特性

### ✅ 异动提醒（可选配置）
- 封涨停 / 打开涨停 / 逼近涨停 / 即将打开涨停
- 封跌停 / 打开跌停 / 逼近跌停
- 新股开板
- 大幅拉升 / 快速跳水
- 板块异动

### ✅ 实时热点题材
显示当前涨幅最大的概念板块

### ✅ 语音播报
使用Windows SAPI语音，稳定可靠

### ✅ 联动通达信
点击异动列表中的股票，自动跳转到通达信对应页面

---

## 数据来源

数据来自**东方财富**实时行情API，无需依赖原版追板精灵。

---

## 依赖安装

首次运行会自动安装，也可手动：
```
py -3 -m pip install pywin32 pillow requests
```

## 文件说明

| 文件 | 说明 |
|------|------|
| `启动追板精灵.bat` | 启动脚本 |
| `zhuiban_app.py` | 主程序 |
| `data_source.py` | 数据获取模块 |

## 注意事项

1. 需要 Python 3.8+
2. 需在交易时间内使用（9:15-15:00）
3. 通达信需要先打开才能联动跳转
